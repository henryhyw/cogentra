# Concentra Demo Data

Concentra seeds three complete assignments into `apps/api/runtime/demo_store.json` and mirrors human-readable artifact files into `demo-data/generated/` when you run:

```bash
pnpm seed
```

The seeded workspace includes:

- `Sustainability Impact Report: Campus Food Delivery`
- `Product Strategy Deck: Habit-Based Budgeting Launch`
- `Technical Notebook: Urban Mobility Demand Analysis`

Each assignment includes at least five student bundles, generated focus points, question sets, and a mix of completed and pending review cases.

For local upload and browser tests, the committed `e2e/` files provide a minimal assignment brief and one student bundle.
